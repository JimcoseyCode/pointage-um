# FDS Pointage UM 🎯

> Une application moderne de gestion d'intérim pour l'Université de Montpellier
### PROJET L3 - INFORMATIQUE

## ✨ Fonctionnalités

- 🔐 **Authentification Sécurisée**
  - Connexion/Inscription [l'inscription sera restreint a l'employeur]
  - Protection des routes
  - Gestion des sessions
- 🎨 **Interface Modern**
  - Design responsive
  - Animations fluides
- 📊 **Gestion d'Intérim**
  - Tableau de bord intuitif
  - Suivi en temps réel
  - Rapports détaillés

## 🚀 Démarrage Rapide

```bash
# Cloner le projet
git clone https://github.com/raphaelrheault/fds-pointage-um.git

# Accéder au dossier
cd fds-pointage-um

# Installer les dépendances
npm install

# Configurer l'environnement
cp .env

# Lancer en développement
npm run dev
```

## 🛠️ Technologies

- **Frontend**: Next.js 13, TypeScript, TailwindCSS, Framer Motion
- **Backend**: Next.js API Routes, Prisma, MongoDB
- **Auth**: NextAuth.js
- **UI**: Shadcn/UI, Lucide Icons
